# Website Color Palette

This document defines the colors used throughout the site, along with 5 variants for each (very-light, light, regular, dark, very-dark).

## 1. Primary Purple
Used for links, selection highlights, active status badges, buttons, and visual focus indicators.
- **Very-Light**: `#f2edff` (Selection background)
- **Light**: `#bfaaff` (Muted details label & back link text)
- **Regular**: `#8859ff` (Primary brand color / button hover fill)
- **Dark**: `#5a30cc` (Hover borders / active highlights)
- **Very-Dark**: `#301080` (Deep shadow borders)

## 2. Solid Background Purple
Used as the solid background color for subpage views like the Time page, Password Generator page, and Pomodoro Timer.
- **Very-Light**: `#4a2c7a`
- **Light**: `#2d1354`
- **Regular**: `#150030` (Core background color)
- **Dark**: `#0c001c`
- **Very-Dark**: `#05000a`

## 3. Dark Gray / Off-Black
Used as background colors for the portal index page cards and container boundaries.
- **Very-Light**: `#444444`
- **Light**: `#2a2a2a`
- **Regular**: `#121212` (Card background / subpage body base)
- **Dark**: `#0d0d0d`
- **Very-Dark**: `#050505`

## 4. Grayish White
Used for text elements, calendar day numbers, and primary header labels.
- **Very-Light**: `#ffffff` (White highlights)
- **Light**: `#f2edff` (Selection text color / base text)
- **Regular**: `#dfdbe5` (Base elements and values text)
- **Dark**: `#b5b0bb` (Subtext and descriptions)
- **Very-Dark**: `#8c8892` (Disabled indicator texts)

## 5. Starbucks Green
Used for the Starbucks Schedule page meta headers, logo context, and brand identification.
- **Very-Light**: `#e6f7f0`
- **Light**: `#33b385`
- **Regular**: `#00704a` (Starbucks brand color)
- **Dark**: `#004d33`
- **Very-Dark**: `#002619`
